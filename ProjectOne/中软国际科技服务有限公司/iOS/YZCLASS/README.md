# 云智课堂iOS端

## 应用于iOS端的学习工具,主要功能:课程学习，观看直播，连麦，聊天室.

## 编译工具 xcode8以上

## 系统版本 iOS9.0以上

## 快速开始

本项目支持cocoapods安装，只需要将如下代码添加到Podfile中：
```
pod 'TXLiteAVSDK_Professional'
```
执行pod install或pod update.
